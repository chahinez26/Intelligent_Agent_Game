# Maze Solving Agents

## Description
This project implements several types of intelligent agents to solve a randomly generated maze.

The agents explore the grid to find a path between a starting position and a target.


## Implemented agents 

- Simple Reflex Agent
- Model-Based Agent (DFS)
- Goal-Based Agent (BFS)
- Utility-Based Agent (A*)
